HTML Cheat Sheet

Forms Tags

Tag Name Description
<input> Text Input Used for single-line text input.
<textarea> Text Area Used for multi-line text input.
<select> Select Creates a dropdown list for selection.
<button> Button Defines a clickable button.
<lable> Label Provides a label for form elements.
<fieldset> Fieldset Groups related elements in a form.
<legend> Legend Defines a caption for the content of a fieldset.
<form> Form Represents an HTML form for user input.
<input type="checkbox"> Checkbox Used for single-line text input.
<input type="radio"> Radio Button Allows users to select a single option from a group.
<input type="submit"> Submit Button Submits the form data to the server.
<input type="checkbox"> Reset Button Resets the form to its default values.
